# 01_Official_Resources

## Folder
Data Cloud

## Purpose
Curated, verified links to official Salesforce Trailhead content, certification guides, architecture documentation, and release notes for Data Cloud / Data 360. Every link below was confirmed live via search at time of writing (June 2026). Salesforce reorganizes Trailhead URLs periodically — if a link 404s, search Trailhead directly for the module/trail title quoted.

## Branding Notes
- All "Data Cloud" branded Trailhead content remains valid post-rebrand; Salesforce states the rename to **Data 360** (October 14, 2025) changed naming and positioning, not underlying functionality.
- Expect both `data-cloud` and `data360`/`data-360` in URL slugs for the next several releases as Salesforce migrates documentation.
